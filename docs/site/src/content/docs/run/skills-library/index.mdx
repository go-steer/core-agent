import { CardGrid, LinkCard, Aside } from '@astrojs/starlight/components';

`core-agent` ships three Claude-Skills–shaped bundles in
[`SKILLS/`](https://github.com/go-steer/core-agent/tree/main/SKILLS)
at the repo root. They teach an agent how to configure and embed
`core-agent` itself — the meta-use case is that `core-agent` can use
them to help a user set up another `core-agent`.

<Aside type="note">
Same material as the static [Agent design](/agent-design/) and
[Concepts](/concepts/) pages, in workflow form. Reading first-time
and want depth? Use those pages. Want an agent to walk you through
setup? Install the relevant skill below.
</Aside>
